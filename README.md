# Умный Дрон-Доставщик (ML-Agents Delivery Drone)

Этот проект представляет собой симуляцию автономного дрона-доставщика, обученного с помощью **Unity ML-Agents**. Дрон умеет самостоятельно находить посылки в сложном городском окружении, забирать их с помощью магнита и доставлять в заданные зоны сброса.

## 🚀 Основные возможности
* **Кастомная физика квадрокоптера**: Дрон использует физику твердого тела (Rigidbody) с реалистичным наклоном (Pitch/Roll) при движении вперед/вбок, а также стабилизацией высоты.
* **Сенсорное зрение (Радар)**: Дрон оснащен `Ray Perception Sensor 3D` (75 лучей видимости), что позволяет ему видеть препятствия (дома), посылки и зоны доставки, и успешно огибать здания.
* **Механика подбора грузов**: Реализован магнитный захват посылки (`PackageDropController.cs`), который срабатывает при приближении к коробке, и сброс посылки при достижении зеленой зоны.
* **Два режима управления**:
  * `Inference Only`: Дрон летает сам, используя натренированную нейросеть (`DroneDelivery.onnx`).
  * `Heuristic Only`: Ручное управление с клавиатуры.

## 🎮 Ручное управление (Клавиатура)
Вы можете переключить любого дрона в режим **Heuristic Only** (в компоненте Behavior Parameters) и летать вручную:
* **W / S** — Наклон вперед / назад (полет вперед/назад)
* **A / D** — Наклон влево / вправо (стрейф боком)
* **Q / E** — Поворот носа влево / вправо (рыскание/yaw)
* **Space / Left Shift** — Газ вверх / Газ вниз (изменение высоты)
* **F** — Включить магнит (взять или бросить коробку)

## 🧠 Процесс обучения (Как это создавалось)
1. **Базовая физика**: Сначала мы написали физику полета и скрипт магнита.
2. **Арены**: Для первичного обучения были созданы небольшие тренировочные арены без препятствий. Дрон научился понимать, что такое коробка и куда её нести.
3. **Перенос в Город**: Мы интегрировали дрона в готовую сцену мегаполиса с домами, дорогами и препятствиями.
4. **Обучение с препятствиями (PPO)**: Были настроены лучевые радары (Raycasts), и запущено параллельное обучение 16 клонов одновременно. Нейросеть методом проб и ошибок научилась доставлять грузы, не разбиваясь о небоскребы.

## 📂 Структура проекта
* `Assets/Scripts/DroneFlightController.cs` — Ядро дрона. Управляет физикой, собирает данные с сенсоров для ИИ, выдает награды (штраф за падение, награда за доставку).
* `Assets/Scripts/PackageDropController.cs` — Управляет захватом и сбросом посылки.
* `Assets/Scripts/CityTrainingManager.cs` — Скрипт телепортации. Случайным образом разбрасывает дрона, посылку и цель по городу в начале каждого эпизода обучения.
* `Assets/Scripts/CityDeliveryManager.cs` — Менеджер для одиночной игры. Позволяет бесконечно спавнить новые коробки и цели после успешной доставки.
* `Assets/ML-Models/DroneDelivery.onnx` — Готовый, натренированный "мозг" (нейросеть) дрона.

## 🛠 Запуск
1. Откройте сцену `demo_city_night`.
2. Убедитесь, что у дрона в поле `Model` компонента `Behavior Parameters` выбран файл `DroneDelivery.onnx`.
3. Нажмите Play в Unity!
